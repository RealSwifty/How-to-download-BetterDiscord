How to install a
theme?
With BetterDiscord
Step 1
Install BetterDiscord
DOWNLOAD
Step 2
Join the BetterDiscord server
JOIN
Step 3
Go in the
#theme-repo
channel, choose a theme and download it.
Step 4
Open discord then go in settings -> BetterDiscord -> Themes -> Open theme folder.
Step 5
Slide the
.theme.css
file you downloaded into the folder that just opened.
Step 6
Go back to Discord and do
ctrl+r
to reload it.
Step 6
Step 6: Go back in settings -> BetterDiscord -> Themes and enable the theme.
Step 7
You are done!
